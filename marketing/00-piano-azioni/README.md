# Piano Azioni Marketing - WasteFlow

Questa cartella contiene il piano operativo completo per il lancio e la crescita di WasteFlow.

## Struttura

- **00-piano-azioni/**: Roadmap e prioritizzazione azioni
- **01-content/**: Guide, video, post social, blog
- **02-sales-materials/**: Template email, LinkedIn, materiali vendita
- **03-website/**: Copy sito, landing pages
- **04-seo/**: Strategia SEO, keywords, content calendar
- **05-partnership/**: Liste contatti, template outreach partner
- **06-analytics/**: Setup tracking, dashboard, KPI

## Quick Start

1. Inizia da `azioni-costo-zero.md` - implementa tutte le azioni gratuite
2. Ogni azione ha link ai file operativi (template, script, copy)
3. Spunta le checkbox quando completi un'azione
4. Traccia i risultati nella sezione KPI di ogni file
