# Lista Keyword SEO Strategiche

Questa lista è la base per la creazione di contenuti (articoli, landing page) e per le campagne Google Ads. Le keyword sono divise per "intention".

### Keyword Transazionali (Intenzione d'Acquisto Alta)

Queste sono le keyword più importanti. L'utente sta cercando attivamente una soluzione da acquistare.

- software rentri
- gestionale rifiuti rentri
- software gestione rifiuti
- programma per formulari rifiuti
- software registro carico e scarico rifiuti
- rentri facile
- prezzo software rentri
- software rentri cloud
- migliore software rifiuti
- software per trasportatori rifiuti
- software per consulenti ambientali

### Keyword Informative (Intenzione di Ricerca Media/Alta)

L'utente sta cercando informazioni. Dobbiamo intercettarlo per diventare la sua fonte di fiducia e poi portarlo alla nostra soluzione.

- cos'è il rentri
- come funziona il rentri
- scadenze rentri 2025
- obbligo rentri
- sanzioni rentri
- come compilare formulario identificazione rifiuti
- vidimazione formulari rifiuti
- registro carico e scarico rifiuti come si compila
- normativa gestione rifiuti
- codice CER (es. "codice cer 170504")

### Keyword "Vs" (Confronto)

L'utente sta comparando soluzioni. Dobbiamo essere presenti.

- [Nome Competitor 1] vs [Nostro Brand]
- alternative a [Nome Competitor 2]
- recensioni [Nome Competitor 1]

**Azione successiva:**
1.  Creare un articolo del blog per ogni keyword informativa principale.
2.  Assicurarsi che la landing page principale sia ottimizzata per le 3-4 keyword transazionali più importanti.
3.  Creare pagine di confronto "vs competitor" per posizionarci come l'alternativa migliore.
