## Section 9: FAQ RENTRI

### Headline (H2)
```
FAQ RENTRI: Le 10 Domande Più Frequenti
```

### Accordion FAQ

**1. Sono obbligato a iscrivermi a RENTRI?**
```
Sì, se produci rifiuti speciali (anche 1kg/anno) da attività produttiva.
Include: manifattura, edilizia, officine, alimentare, commercio.

Esclusi: solo rifiuti urbani assimilati gestiti da Comune.
```

**2. Entro quando devo iscrivermi?**
```
Obbligo: febbraio 2026.

Ma parti ORA perché:
• Certificato digitale: 30-60gg
• Iscrizione RENTRI: 15gg lavorativi
• Formazione team: 2-4 settimane
• Test sistema: 1-2 mesi

Chi inizia a gennaio 2026 sarà in ritardo.
```

**3. Quanto costa iscriversi a RENTRI?**
```
Iscrizione RENTRI = GRATIS (portale nazionale).

Paghi solo:
• Certificato firma digitale: 30-50€/anno
• Software gestione (optional): 49-300€/mese

WasteFlow include tutto a 49€/mese.
```

**4. Posso ancora usare FIR cartacei dopo feb 2026?**
```
No. Solo digitali.

Eccezione: blackout sistema RENTRI nazionale (dovrai documentare).

Sanzione FIR cartaceo: 1.600-9.300€ cad. ⚠️
```

**5. Cosa succede se sbaglio un CER?**
```
CER errato = rifiuto non tracciabile = sanzione 1.000-10.000€.

WasteFlow valida in tempo reale:
• CER esistente e attivo
• Compatibile con tua attività (ATECO)
• Destinatario autorizzato per quel CER

Alert immediato se c'è problema.
```

**6. Come gestisco i FIR se autisti non hanno smartphone?**
```
3 opzioni:
1. Smartphone aziendale dedicato (30-50€)
2. Tablet in cabina camion
3. Compilazione ufficio + firma OTP differita

Con WasteFlow: app leggerissima, funziona anche su Android 8 (2017).
```

**7. RENTRI offline, cosa faccio?**
```
Protocolo emergenza:
1. FIR cartaceo temporaneo
2. Entro 48h: digitalizza su RENTRI appena online
3. Allega nota blackout

WasteFlow: modalità offline → salva in locale → sync automatico quando torna rete.
```

**8. Il mio consulente può gestire RENTRI per me?**
```
Legalmente: responsabilità è del produttore rifiuti (TU).

Praticamente: consulente può operare per te (con procura o accesso delegato).

WasteFlow piano consulenti: gestisci 10-50 clienti da dashboard unica.
```

**9. Quanto tempo ci vuole per un FIR digitale?**
```
Portale RENTRI base: 5-10 minuti/FIR

WasteFlow:
• Primo FIR: 3-5 minuti (learning)
• A regime: 30 secondi (QR scan → selezione → done)

Risparmio: 80-90% tempo.
```

**10. E se cambio software dopo?**
```
Export completo dati (CSV, PDF).
RENTRI mantiene storico nazionale (è il tuo backup legale).

WasteFlow: zero lock-in. Cancelli in 2 click. Nessuna penale.
```
